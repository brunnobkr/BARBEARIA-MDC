# Pasta de Assets

Coloque a imagem do logo da Barbearia MDC aqui.

**Nome do arquivo:** `logo.svg` ou `logo.png`

**Formato aceito:** 
- SVG (recomendado para melhor qualidade)
- PNG com fundo transparente
- JPG

**Tamanho recomendado:** 
- Largura: 400-600px
- Altura: proporcional

**Como adicionar:**
1. Coloque sua imagem do logo nesta pasta (`src/assets/`)
2. Nomeie o arquivo como `logo.svg` ou `logo.png`
3. Se usar outro nome, atualize os imports em:
   - `src/components/Navbar.tsx`
   - `src/pages/Home.tsx`

O logo será exibido automaticamente na navbar e na página inicial após você adicionar o arquivo.

**Nota:** Atualmente há um placeholder SVG temporário. Substitua-o pela sua imagem real.
